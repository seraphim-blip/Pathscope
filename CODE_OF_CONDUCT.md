# Code of Conduct

## Our Commitment
The Pathscope community is committed to fostering a professional, respectful, and technically constructive environment.

Contributors, maintainers, and users are expected to collaborate in ways that improve the project’s reliability, maintainability, and responsible use.

---

## Expected Behavior
Please:

- communicate respectfully and clearly
- focus on technical accuracy
- provide actionable feedback
- keep discussions solution-oriented
- respect maintainer decisions
- prioritize safe and authorized use cases
- document assumptions when reporting issues

---

## Unacceptable Behavior
The following is not acceptable:

- harassment or personal attacks
- abusive or insulting language
- spam or disruptive behavior
- intentionally misleading technical claims
- irresponsible usage guidance
- contributions designed for unauthorized access
- sharing sensitive data in public issues

---

## Scope
This policy applies to:

- GitHub issues
- pull requests
- discussions
- documentation contributions
- community support channels
- release notes discussions

---

## Enforcement
Maintainers may remove, edit, or reject contributions that violate this policy.

Repeated violations may result in restricted repository participation.

---

## Responsible Collaboration
Pathscope is intended for **authorized internal security auditing and engineering visibility workflows only**.

Contributions that conflict with this intent may be declined.
